Literal value of a function:
\x -> x * x - 1

Value between \ and -> is the parameter of the function. It can be anything. When function is applied, all occurrences of the parameter are replaced by a given argument.

twiceMinusOneA x = x*x -1 -- Which is the same as twiceMinusOneB = \x -> x*x -1

myFunction = \x -> x * x - 1
twice = \f -> (\x -> f (f x))

Evaluate twice myFunction 5

--------------------------

twice = \myFunction -> (\x -> myFunction (myFunction 5))
twice = \myFunction -> (\x -> myFunction (24))
twice = \myFunction -> (\x -> 575)
